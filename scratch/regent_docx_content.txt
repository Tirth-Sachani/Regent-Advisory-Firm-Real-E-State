2. Server Route Handler (`app/api/ai/recommendations/route.ts`):
   - Securely read the incoming user session ID via Supabase server-side client components.
   - Query the `user_preferences` table to pull recent query parameters and choices.
   - Use the official Google Gen AI SDK (`@google/genai`) to pass user preference text strings into an embedding model to extract a 1536-dimension text vector.
   - Perform a vector cosine similarity match against the `properties.property_embedding` column using a standard threshold query filter.
   - Log the recommended property match history inside the `recommendation_history` relational datastore.
3. Code Quality: Enforce complete TypeScript interface checking, return structured JSON objects, and avoid front-end logic coupling.
